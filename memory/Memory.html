Memories에 오신 것을 환영합니다

마크다운으로 학습 자료를 만들고, 게임으로 복습하세요

1

학습 자료 만들기

좌측 메뉴에서 "새 파일"을 클릭하거나, 로컬 MD 파일을 추가하세요. **단어** 형식으로 중요한 단어를 표시하면 게임에서 빈칸 문제로 출제됩니다.

2

게임 모드 선택

좌측 메뉴의 "설정"에서 게임 모드를 선택하세요.
• 무작위 모드: 잘 안 쓰이는 어려운 단어 중심 출제
• TTV 모드: **굵게 표시**한 단어만 출제

3

게임 시작

좌측 메뉴에서 파일을 클릭하면 바로 게임이 시작됩니다.
• 마이 파일: Supabase 클라우드에 저장된 파일
• 로컬 파일: 디바이스에서 불러온 파일

💡 Tip: 표 형식도 지원됩니다!

엑셀에서 표를 복사해서 붙여넣으면 자동으로 마크다운 표로 변환됩니다.
표 안의 **굵은 글씨**도 빈칸 문제로 출제됩니다.
